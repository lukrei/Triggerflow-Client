Triggerflow-Client<br>

A Triggerbot for CS2 using memflow.<br>
This is a fork of https://github.com/superyu1337/radarflow2. 
Clone the repo on your vm:
git clone https://github.com/lukrei/Triggerflow-Client.git 
Then Update Offset:
./update.sh
Compile it again:
./rebuild.sh 
Run the program:
./run.sh
